# Captures d'écran Keydy

Place les vraies captures du programme dans ce dossier avec ces noms pour que le site les affiche automatiquement:

- `bureau.png`
- `tablette-technicien.png`
- `devis-factures.png`
- `planning.png`
- `chantiers.png`

Les mêmes noms en `.jpg` fonctionnent aussi. Pour garder le site propre, utilise des captures larges pour `bureau.png` et des captures verticales ou tablette pour `tablette-technicien.png`.
