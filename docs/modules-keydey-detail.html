Planning / Calendrier
+5 €/utilisateur/mois HTVA
Planifiez les rendez-vous de vos techniciens, visualisez leur charge, détectez les conflits et gérez les absences. Le tout en temps réel.
Ce que comprend ce module
✓
Vue mois / semaine / jour — Navigation fluide, code couleur par type d'intervention
✓
Multi-techniciens — Filtres par employé, vue d'ensemble de l'équipe
✓
Création RDV rapide — Type, client, site, créneau, description en un écran
✓
Détection de conflits — Avertissement visuel si un technicien est déjà occupé
✓
Sélection intelligente du créneau — Visualisation des disponibilités par quart d'heure
✓
Gestion des absences — Congés, maladie, formation visibles sur le planning
✓
Récurrence — RDV multi-jours pour les gros chantiers
✓
Synchronisation mobile — Les RDV apparaissent instantanément sur KeydyApp
Bon à savoir : Le planning est automatiquement alimenté par les contrats de maintenance (module SAV) qui génèrent des interventions préventives.

SAV / Contrats / Attestations / F-Gas
+8 €/utilisateur/mois HTVA
Tout le suivi après-vente : contrats de maintenance, attestations réglementaires, cycle de vie des équipements et journal F-Gas.
Ce que comprend ce module
✓
Contrats de maintenance — Fréquence configurable (mensuel à biennal), facturation auto
✓
Planning préventif — Interventions créées automatiquement N jours avant échéance
✓
Cycle de vie équipements — Âge, garantie, durée de vie estimée, alertes remplacement
✓
Attestations d'entretien — Formulaire officiel pré-rempli, PDF conforme
✓
Attestations mise en service — PV officiel avec données techniques
✓
Journal F-Gas — Suivi des fluides frigorigènes, conformité réglementaire
✓
Alertes par email — Équipements à remplacer, garanties expirantes, contrats à renouveler
✓
Forfaits entretien intelligents — Auto-suggestion du bon forfait selon l'installation, DZ incluse
Obligatoire : Pour les entretiens (mazout, gaz, pellet), l'attestation officielle doit être générée avant de pouvoir clôturer l'intervention.
